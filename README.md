# Random Password Generator

A web-based tool for generating strong, unique passwords with customizable options.

## Features

- **Configurable password length** (6-24 characters)
- **Character type toggles**:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Symbols (!@#$%^&\*...)
- **Password strength indicator** with humorous feedback
- **Copy to clipboard** functionality
- **One-click password regeneration**

## Usage

Simply open `index.html` in a web browser. Adjust the password length using the slider, toggle the desired character types, and click the generate button to create a random password.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome (icons)
